CSS Properties Demo

Font-Size

Absolute Units

1 px is 1/96th of an inch, not the pixel of the screen of the viewing device Don't have spaces in-between the values and the CSS unit like this: "12 px", rather "12px"

Half a centimetre font-size.

Relative Units

em is relative to the font-size of the parent element if we're using it for font properties. If we're using em for anything that's not font related, then its relative to this element's font-size. The element's font-size is 0.5em x 40px = 20px The border (non-font property) is: 0.25em x (0.5em x 40px) 0.25em x 20px 5px

If the root element's (html element) font-size is 16px, then 1.5rem x 16px = 24px
